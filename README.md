# sim
Simulated different betting strategies on real set of up to 12000 consecutive roulette wheel results.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
